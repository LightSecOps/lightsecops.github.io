T1173 - Dynamic Data Exchange (DDE)

ID: T1173 Tactique: Exécution Plateforme: Windows Autorisations requises: Utilisateur Sources de données:
Version: 1 Création: 17/03/2020 Dernière modification: 17/03/2020

Description

Le protocole DDE est une ancienne technologie de Microsoft utilisée pour facilite le transfèrer des données entre deux applications. Inter-Process Communication (IPC).

Cette technique peut être utilisé dans les scénarios de Phishing.

l’exécution du code dans les documents Microsoft Office.
